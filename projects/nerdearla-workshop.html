Workshop Overview

A comprehensive workshop presented at NerdearLA 2025, demonstrating the evolution of AI agent workflows using cutting-edge technologies. This hands-on session guided developers through building sophisticated, modular AI agent systems from basic responses to advanced multi-server deployments.

The workshop showcased practical implementation of modern AI development patterns, emphasizing scalability, modularity, and real-world application scenarios.

Workshop Details

  • Event: NerdearLA 2025
  • Date: March 23, 2025
  • Duration: 1 hour 10 minutes
  • Format: Live coding demonstration
  • Audience: AI developers and enthusiasts

Workshop Modules

Progressive AI Agent Development Journey

Module 1: Foundation

Simple AI Responses - Basic interaction patterns and response generation

Module 2: Agent Patterns

ReAct Agent Implementation - Reasoning and Acting frameworks for intelligent agents

Module 3: Architecture

MCP Server Architecture - Model Context Protocol for scalable AI systems

Module 4: Workflows

Custom LangGraph Workflows - Building complex agent interaction patterns

Module 5: User Experience

Interactive Permission Systems - Human-in-the-loop agent controls

Module 6: Integration

Tool Integration Techniques - Connecting agents with external services

Module 7: Deployment

Multi-Server Deployment - Scaling AI agents across distributed systems

Technical Implementation

Core Technologies

  • LangChain - Framework for developing applications with large language models
  • LangGraph - Library for building stateful, multi-actor applications with LLMs
  • Model Context Protocol (MCP) - Standardized protocol for AI model interactions
  • Gradio - Interactive machine learning interface for demonstrations
  • SQLite - Lightweight database for practical examples and data persistence

Advanced Concepts Covered

  • State Management: Handling complex agent states across interactions
  • Tool Integration: Connecting AI agents with external APIs and services
  • Permission Systems: Implementing human oversight and approval workflows
  • Multi-Agent Coordination: Orchestrating multiple AI agents working together
  • Scalable Architecture: Designing systems that grow with demand

Learning Outcomes

Participants gained hands-on experience in:

  • AI Agent Design Patterns: Understanding when and how to apply different agent architectures
  • Modern AI Frameworks: Practical experience with LangChain and LangGraph
  • Production Considerations: Scaling, monitoring, and maintaining AI systems
  • User Experience Design: Creating intuitive interfaces for AI interactions
  • System Integration: Connecting AI agents with existing infrastructure
  • Best Practices: Industry standards for AI agent development
